wsds

Simple input overlay for use with the input-overlay plugin for OBS Studio.

Overlays originally for use on Tetris gameplays and for emulation, then expanded for bullet hell/UNDERTALE and DELTARUNE.

Variants

  • WASD Layout, based on my Tetris control setup
  • WASD Layout without spacebar and "I" key
  • Arrow keys layout, for games that use this control scheme (Undertale/Deltarune)
  • No mouse variant in separate layout config (nomouse.json) in WASD and Arrowkey layout
